# Python ship game

This is a 2D space shooter game developed in Python3 using the `pygame` library. The player controls a spaceship, shoots bullets, and fights off aliens.

## Requirements

- Python 3.x
- Pygame library

Install the dependencies with:
pip install pygame

## Game files
1. alien_invasion.py  
2. bullet.py  
3. game_functions.py  
4. settings.py  
5. ship.py  

## How to Play
Run the command:  
python alien_invasion.py
